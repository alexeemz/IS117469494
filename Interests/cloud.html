What is the Cloud?
Cloud Computing is the term used to describe the provision of services and access of data over the internet.

What's the point?
There are many benefits to Cloud Computing!

  • In the past people would run programs from software downloaded onto the hardrive of their own personal computer.

  • Through the cloud people can now access the same programs over the internet which makes software much more available and efficient!

  • Aswell as easier access to more programs, computing over the cloud means that you can work from anywhere and everywhere as all of your data is stored online and can be accessed from any computer connected to the internet!

Software as a Service(SaaS)

Sofware as a service (Commonly referred to as SaaS) is a method of distributing software to clients over the internet on a subscription basis and encompasses PaaS, IaaS and Caas
which are described in subsequent pages.
The great thing about SaaS is that the user does not have to worry about installing and maintaining the software as the software provider manages the application while the user
accesses it simply through a web browser.

Advantages:

  • Lower Initial costs as no installation or hardare required.
  • Seamless Integration.
  • Painless Upgrades.

Platform as a Service(PaaS)



Platform as a Service (Commonly referred to as PaaS) is a cloud computing model in which providers deliver a development environment over the internet to clients who pay on a pay-as-you-go basis.
Just as with SaaS, no installation or hardware is required which lessens the expense and complexity for users.

Advantages:

  • PaaS tools can cut coding time with pre-coded components which are built in to the platform.
  • The Pay-as-you-go model makes it very affordable for developers to use sophisticated development tools which would be very expensive otherwise.
  • Paas supports teams working in different parts of the world because the development environment is accessed through the internet.

An example of PaaS is Cloud9 IDE.
Cloud9 is an integrated development environment which allows coders to begin coding immediately with pre-configured workspaces and provides many features such as live previews of the project.

Infrastructure as a Service(IaaS)



Just like SaaS and PaaS, Infrastructure as a Service provides access to a computing resource through the internet.

IaaS itself specifically refers to 'virtualised hardware' which users can make use of without physically needing access to the otherwise necessary equipment. Services provided by IaaS include storage, networking and netorking services such as firewalls.

Amazon Web Services (AWS) is one of the leading IaaS providers, offering a huge range of infrastructural development tools on a pay-as you-go basis.

Communication as a Service(CaaS)



Communication as a Service, or CaaS, is a term that encompasses all communications services which are outsourced by enterprises and provided through the internet.
CaaS could include voice and video calls, conference calls, instant messaging and much more.
Just like other forms of SaaS, CaaS is attractive to businesses as no outright ownership costs are required for the use of software and applications. Instead Payment is based on consumption on a pay-as-you-go basis.

Gmail is a common example of Communication as a Service, most commonly known as a free email service.
